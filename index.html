Road Trip Weather Forecast

This app is here to help you to get weather forecast for a journey with a complex route. Think Road Trip where you stay at a different location each day.

Simply pick the start date of your journey and it's duration. Enter planned stops for each day of your journey and get weather for this particular stop on a given day.

Note that this app currently restricts number of stops to the maximum of 10.

Weather forecast is powered by Dark Sky. Dark Sky API provides 7-day weather forecast and historical data where forecast is not available.
